GitHub Gist Sync Settings

Setup Instructions:

  1. Create a GitHub Personal Access Token:
    • Check the gist scope checkbox
    • Click "Generate token" at the bottom
    • Copy the token (starts with ghp_)
  2. Paste the token below and set a push password
  3. Click Save - the app will automatically create a private Gist for your words
  4. Leave Gist ID empty on first setup (it will be filled automatically)

Note: All family members can share the same GitHub token and Gist. Each user's word list will be stored in a separate file (e.g., kai-words.json, mom-words.json) within the same Gist. Share the token and password with family members so everyone can sync across their devices.
